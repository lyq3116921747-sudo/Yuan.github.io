你好，我是 Yuan。这个博客用于记录技术学习、项目实践和一些值得反复回看的想法。

目前站点使用 Jekyll 构建，文章放在 `_posts` 目录中。以后只需要新增 Markdown 文件并写好 front matter，就可以通过 GitHub Pages 自动发布。

## 写作方向

- 技术问题的排查过程
- 项目实践和复盘
- 工具、阅读和学习笔记

## 文章模板

```markdown
---
title: 文章标题
date: 2026-05-05 12:00:00 +0800
categories: [技术]
tags: [Jekyll, Blog]
description: 一句话概括这篇文章。
---

正文从这里开始。
```
